About

I code in order to find and understand patterns in data that quantify geologic structures.

Right now I'm finishing work that aims to compare observations with randomness, using data that quantify discrete structures along a line in space or time.

Mostly for my own benefit, I wrote Matlab code that lets you interrogate your own data using the same tools, so I created an app that runs on Mac and PC.
